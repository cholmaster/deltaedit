### How do I get set up? ###
*Dependency
pygobject
webkitgtk(gir1.2-webkit-3.0)
python3-pip
* Setup
Linux:Run Installer.py to install, run Uninstaller.py to uninstall.
* How to run
	dedit(Text Editor):
			Korean:type dedit-ko
			Chinese:type dedit-cn
			Japanese:type dedit-jp
	gmemo(Memo App):gmemo
